# Siliqua

Add a multiude of new actions to your AirPods.

Set multiple actions for a double tap and a quad tap.

## Actions

* Pause/Play.

* Skip.

* Rewind.

* Skip15.

* Rewind15.

* IncreaseVolume.

* DecreaseVolume.

* ToggleSiri.
